# Kasdex Agent Entry Point

This repository contains the `kasdex` source code.

Before doing any work in this repo, read and follow:

- `.harness/AGENTS.md`

The `.harness` directory contains local agent workspace files, including additional rules, skills, commands, notes, and working artifacts. Source code belongs in the repository root; agent workspace material belongs under `.harness`.

If `.harness/AGENTS.md` is missing, stop and ask the user to restore the local agent workspace before making non-trivial changes.
